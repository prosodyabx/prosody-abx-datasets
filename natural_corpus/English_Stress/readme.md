# English Natural Corpus (Lexical Stress in Noun/Verb Words)

This directory contains the natural (human-recorded) English corpus used in the Prosodic ABX experiments.

## Overview

- Speakers: 10 native speakers of American English 
- Target words: 15 English words that can be used as both nouns and verbs
- Goal: collect noun/verb stress realizations of the same word from the same speaker

In dictionary pronunciations, these words often show different stress patterns across parts of speech. The recording design is built to capture that contrast.

## Experimental Design

For each target word:

- The noun usage is elicited with 2 sentence prompts
- The verb usage is elicited with 2 sentence prompts
- Total: 4 sentence conditions (`sid = 01-04`)

This setup allows comparison of how one speaker pronounces different stress patterns of the same lexical form under different grammatical roles.

## Directory Structure

## OSF Upload Note

Audio files (`utterances` and `segments`) are uploaded as `.zip` archives because uploading a very large number of individual files to OSF is slow and may cause upload failures.

Please unzip the archives before use.  
The metadata files (`metadata/*.tsv`) reference utterance/segment IDs by file name.

## Audio Files

- Sample rate: 48 kHz

### `audio/utterances/`

Full sentence recordings.

Filename format:

```text
en_{speaker}_{contrast_index}_{sid}.wav
```

Example:

```text
en_S001_c01_01.wav
```

- `speaker`: speaker ID (e.g., `S001`)
- `contrast_index`: target-word ID (e.g., `c01`, `c02`, ...)
- `sid`: sentence condition ID (`01`-`04`)

### `audio/segments/`

Target-word clips extracted from utterances.

Filename format:

```text
en_{speaker}_{contrast_index}_{sid}_seg.wav
```

Example:

```text
en_S001_c01_01_seg.wav
```

Word boundaries were first generated by MFA and then manually adjusted.

## Metadata

### `metadata/utterances.tsv`

Sentence-level metadata.

- `utterance_id`: full-sentence audio ID (without extension)
- `speaker`: speaker ID
- `contrast`: target word (e.g., `abstract`)
- `word`: surface word in the sentence
- `label`: stress label (which syllable carries lexical stress)
- `text`: sentence text

`label` is annotated by a native speaker and indicates the index of the stressed syllable.

### `metadata/segments.tsv`

Segment-level metadata.

- `segment_id`: clipped segment ID
- `utterance_id`: parent utterance ID
- `onset`: segment start time (s)
- `offset`: segment end time (s)
- `duration`: segment duration (s)

### `metadata/recording_design.tsv`

Speaker-independent recording design.

- `contrast_index`: target-word ID (`c01`, `c02`, ...)
- `contrast`: target word
- `sid`: sentence condition ID (`01`-`04`)
- `word`: target word in text
- `text`: prompt sentence
